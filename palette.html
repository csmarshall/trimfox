trimfox — palette explorer

Every trimfox rule routes through these --tf-* primitive tokens, so a whole theme is just this token set. Pick a palette to preview it live — the swatches and the browser-chrome mockup both re-color from the same variables. Each maps to a file in chrome/palettes/; set it by editing the one @import line in userChrome.css (light/dark auto-follows macOS appearance).
